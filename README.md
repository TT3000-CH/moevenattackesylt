# Möwenpik V1

Ein kleines browserfähiges Comic-Game für GitHub Pages.

## Spielidee
Du spielst eine Möwe auf Sylt. Ziel ist, Glace und Burger von Touristen zu klauen und möglichst viele Punkte zu sammeln, ohne von den Touristen abgewehrt zu werden.

## Steuerung
- Mobile: Tippen
- Desktop: Klick, Leertaste oder Pfeil nach oben

## Punkte
- Glace: 10 Punkte
- Burger: 15 Punkte
- Tourist berührt: 1 Leben weniger

## Lokal starten
Einfach `index.html` im Browser öffnen.

## Auf GitHub Pages publizieren
1. Neues Repository erstellen, z.B. `moewenpik`.
2. Die Dateien `index.html`, `style.css`, `game.js` und `README.md` direkt ins Repository hochladen.
3. In GitHub: `Settings` > `Pages`.
4. Source: `Deploy from a branch`.
5. Branch: `main`, Ordner: `/root`.
6. Speichern und den angezeigten GitHub-Pages-Link öffnen.
